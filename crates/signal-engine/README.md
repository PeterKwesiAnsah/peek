# signal-engine

Signal impact analysis for the [Peek](https://github.com/ankittk/peek) process intelligence tool.

Describes what happens when you send a signal (e.g. SIGTERM, SIGKILL) to a process: systemd unit detection, TCP connections, child processes. Used by the peek CLI `--kill` panel.

Part of the [peek](https://github.com/ankittk/peek) workspace.
